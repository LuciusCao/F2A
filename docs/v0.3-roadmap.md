# F2A v0.3 功能设计：Agent 协作网络

## 概述

将 F2A 从"配对工具"升级为"Agent 协作网络"，支持 Agent 之间的消息通信、技能调用和文件分享。

## 核心功能

### 1. 消息通信 💬

Agent 之间实时发送文本消息，支持一对一和群组聊天。

```javascript
// 发送消息给 peer
await f2a.sendMessage(peerId, "帮我 review 这段代码");

// 接收消息事件
f2a.on('message', ({ from, content, timestamp }) => {
  console.log(`收到来自 ${from} 的消息: ${content}`);
});
```

**消息格式**:
```json
{
  "type": "message",
  "id": "msg-uuid",
  "from": "agent-a-uuid",
  "to": "agent-b-uuid",
  "content": "消息内容",
  "timestamp": 1709000000000,
  "signature": "签名"
}
```

### 2. 技能调用 🛠️

查询 peer 的可用 skills，并远程调用执行。

```javascript
// 查询 peer 的技能列表
const skills = await f2a.querySkills(peerId);
// [{ name: 'weather', description: '获取天气' }, ...]

// 远程调用技能
const result = await f2a.invokeSkill(peerId, 'weather', {
  location: '北京'
});
// { temperature: 25, condition: 'sunny' }
```

**技能查询请求**:
```json
{
  "type": "skill_query",
  "requestId": "req-uuid"
}
```

**技能查询响应**:
```json
{
  "type": "skill_response",
  "requestId": "req-uuid",
  "skills": [
    {
      "name": "weather",
      "description": "获取天气信息",
      "parameters": {
        "location": { "type": "string", "required": true }
      }
    }
  ]
}
```

**技能调用请求**:
```json
{
  "type": "skill_invoke",
  "requestId": "req-uuid",
  "skill": "weather",
  "parameters": { "location": "北京" }
}
```

**技能调用响应**:
```json
{
  "type": "skill_result",
  "requestId": "req-uuid",
  "status": "success",
  "result": { "temperature": 25, "condition": "sunny" }
}
```

### 3. 文件分享 📁

安全地分享文件给 peer，支持断点续传和完整性校验。

```javascript
// 发送文件
const fileId = await f2a.sendFile(peerId, '/path/to/file.pdf');

// 接收文件事件
f2a.on('file', ({ from, fileId, filename, size, stream }) => {
  // 处理接收的文件流
});

// 查询文件传输状态
const status = await f2a.getFileStatus(fileId);
```

**文件传输流程**:
```
1. 发送方: 发送文件元数据 (文件名、大小、MD5)
2. 接收方: 确认接收或拒绝
3. 发送方: 分块发送文件数据
4. 接收方: 校验完整性，确认完成
```

**文件元数据**:
```json
{
  "type": "file_offer",
  "fileId": "file-uuid",
  "filename": "document.pdf",
  "size": 1048576,
  "md5": "d41d8cd98f00b204e9800998ecf8427e",
  "chunks": 10
}
```

## 协议扩展

### 消息类型

```javascript
const MessageType = {
  // 基础协议
  IDENTITY: 'identity',
  PEER_CONNECTED: 'peer_connected',
  
  // 消息通信
  MESSAGE: 'message',
  MESSAGE_ACK: 'message_ack',
  
  // 技能调用
  SKILL_QUERY: 'skill_query',
  SKILL_RESPONSE: 'skill_response',
  SKILL_INVOKE: 'skill_invoke',
  SKILL_RESULT: 'skill_result',
  
  // 文件传输
  FILE_OFFER: 'file_offer',
  FILE_ACCEPT: 'file_accept',
  FILE_REJECT: 'file_reject',
  FILE_CHUNK: 'file_chunk',
  FILE_COMPLETE: 'file_complete',
  
  // 心跳
  PING: 'ping',
  PONG: 'pong'
};
```

### 安全机制

1. **消息签名**: 所有消息使用发送方私钥签名，接收方用公钥验证
2. **端到端加密**: 配对后协商会话密钥，后续通信加密
3. **防重放攻击**: 消息包含时间戳和唯一 ID

## 使用场景

### 场景 1: 代码审查协作

```javascript
// Agent A 发送代码给 Agent B 审查
await f2a.sendMessage(peerB, '请帮我 review 这个函数');
await f2a.sendFile(peerB, './src/utils.js');

// Agent B 审查后返回意见
await f2a.sendMessage(peerA, '第 23 行有个潜在 bug...');
```

### 场景 2: 技能共享

```javascript
// Agent A 查询 Agent B 的技能
const skills = await f2a.querySkills(peerB);

// 发现 B 有 "translate" 技能，调用它
const result = await f2a.invokeSkill(peerB, 'translate', {
  text: 'Hello World',
  targetLang: 'zh'
});
```

### 场景 3: 分布式任务

```javascript
// 将任务分发给多个 Agent
const peers = f2a.getConnectedPeers();
const results = await Promise.all(
  peers.map(peer => 
    f2a.invokeSkill(peer.agentId, 'analyze', { data: chunk })
  )
);
```

## 实现优先级

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P0 | 消息通信 | 基础功能，实现简单 |
| P0 | P2P 连接 | WebRTC 直连或 Relay 回退 |
| P1 | 技能调用 | 需要设计技能注册/发现机制 |
| P2 | 文件分享 | 涉及分块传输和完整性校验 |
| P3 | 端到端加密 | 安全增强 |
